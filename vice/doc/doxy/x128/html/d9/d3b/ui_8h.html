◆ ui_create_main_window()

void ui_create_main_window ( video_canvas_t *  canvas)

Create a toplevel window to represent a video canvas.

This function takes a video canvas structure and builds the widgets that will represent that canvas in the UI as a whole.

While it creates the widgets, it does not make them visible. The video canvas routines are expected to do any last-minute processing or preparation, and then call ui_display_main_window() when ready.

Parameters
[in]canvasthe video_canvas_s to initialize
Warning
The order of the windows created for x128 depends on the order of the calls to vicii_init() and vdc_init() in src/c128/c128.c. That order is currently set to vicii before vdc so we get the proper window indexes. There has to be a better way.
