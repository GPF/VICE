◆ SCALE_CSS

#define SCALE_CSS
Value:
"scale value {\n" \
" font-family: monospace;\n" \
" font-size: 90%;\n" \
"}"

CSS applied to scale to fix up value display.

Fixed width font to get proper alignment of the value display of the sliders, slightly reduced font size to try to keep it roughly the same size as the sans-serif fonts used in the Adwaita theme (on Gnome Shell anyway).
