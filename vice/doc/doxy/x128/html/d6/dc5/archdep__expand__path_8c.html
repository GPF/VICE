Generate heap-allocated full pathname of orig_name.

Returns the absolute path of orig_name.

Expands '~' to the user's home path on Unix. If the prefix in orig_name is not '\' and not '/' and not '~/' (Unix) the file is assumed to reside in the current working directory, whatever that may be.
