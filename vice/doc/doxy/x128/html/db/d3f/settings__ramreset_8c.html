CSS for the preview of the pattern.

◆ create_preview_widget()

static GtkWidget * create_preview_widget ( void  )
static

Create textview for the RAM init preview hexdump.

We use a textview here and not a label since a textview communicates with its scrolled winded to determine what to render and what not. We used to render inside a label, which had the advantage of keeping the scrolled window at the position the user had scrolled to when updating the contents. The downside is that the view now scrolls back to the top when changing contents. But keep in mind it took 5-8 seconds on a Win10 VM with 2 cores at 3GHz to render, making the UI unresponsive during this time.
