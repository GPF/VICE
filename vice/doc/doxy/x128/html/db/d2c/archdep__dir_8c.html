◆ INITIAL_ARRAY_SIZE

#define INITIAL_ARRAY_SIZE   64

Initial size of an array of file/directory entries.

Number of elements in the arrays when initially allocating them.

This number was chosen arbitrarily, reasonable trade-off between wasting memory for small directories and not triggering too many realloc calls with larger directories.
