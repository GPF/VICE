◆ UIDATA_ROOT_PATH

#define UIDATA_ROOT_PATH   "/org/pokefinder/vice"

Root namespace of the GResouce files.

The GResource handling requires setting a namespace for the files, which is similar to the argument to gtk_application_new(), basically inverting a URL's components.

In this case, I have used vice.pokefinder.org as our website.
