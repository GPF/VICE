Detailed Description

Widget to show a list of radio buttons for RAM sizes.

Creates a grid with title and a vertically oriented radio group with radio buttons with KiB/MiB/GiB labels for each ram size given. Note that the sizes are taken to be in kibibytes, so an integer value of 64 will result in "64KiB" and the resource value 64 (NOT 65536).

Create radio group with RAM size entries and a title.

Create a GtkGrid with a label in the top row set to title and a resource radio group for sizes. If title is NULL a resource radio group is returned without the surrounding grid.

The sizes are taken as kibibytes, so plain 64 results in "64KiB" while setting the resource to 64 (not 65536).
