void video_canvas_adjust_aspect_ratio ( struct video_canvas_s *  canvas)

React to changes of aspect ratio in the physical system.

Rescale and reposition the screen inside the canvas if the screen's size has been programatically changed.

For GTK3 this means we need to compute a new minimum size for our display windows.
