Essentially, this is the same info as acia_irq_type. As the resource stored in the VICE system is different from the actual value in acia_irq_type, the resources value is stored here.

◆ irq_type

enum cpu_int acia_struct::irq_type

the type of interrupt implemented by the ACIA

The ACIA either implements an IRQ (IK_IRQ), an NMI (IK_NMI), or no interrupt at all (IK_NONE).

Note
As some cartridges can be switched between these modes, it is necessary to remember this value.
