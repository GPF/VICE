Detailed Description

VIC20 memory expansion widget.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl

Allow enabling/disabling VIC-20 RAM expansions, and also provides a list of common memory expansion configurations.
