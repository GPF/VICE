void kbd_debug_widget_update ( GtkWidget *  widget,

Update keyboard debug widget with event.

Updates the labels in widget with information from event. Only if widget is the primary instance will the text buffers be updated and scrolled. For all instances the label texts will be updated with the text buffers' contents.

[in]widgetkeyboard debug widget instance
