Detailed Description

CPU speed, FPS display, Pause, Warp widget for the statusbar.

Widget for the status bar that displays CPU speed, FPS and warp/pause state.

When primary-button-clicking on the widget a menu will pop up allowing the user to control refresh rate, emulation speed, warp and pause.
