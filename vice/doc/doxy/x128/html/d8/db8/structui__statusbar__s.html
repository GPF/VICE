Detailed Description

The full structure representing a status bar widget.

This includes the top-level widget and then every subwidget that needs to be individually addressed or manipulated by the status-report API.

◆ widget_row_column

int ui_statusbar_s::widget_row_column

Current column to append new widgets.

Keeps track of the column to append new widgets with statusbar_append() and statusbar_apppend_end().

The function statusbar_append() appends widgets from left to right to the top row of a status bar, the function statusbar_append_end() is used to put the last top widget (volume) on the status bar setting its aligment and hexpand properties so it appears to the right of the status bar.

Referenced by statusbar_append_widget(), statusbar_append_widget_end(), and ui_statusbar_init().
