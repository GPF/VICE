Detailed Description

Retrieve home directory of current user.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl

Retrieve the home directory of the current user on systems that have a concept of a home directory. On systems that don't have a home dir '.' is returned (PROGDIR: in the case of AmigaOS). Of course on systems that don't have a home dir, this function simply shouldn't be used, archdep_boot_path() might be a better function.

OS support:

  • Linux
  • Windows
  • MacOS
  • BeOS/Haiku (always returns '/boot/home')
