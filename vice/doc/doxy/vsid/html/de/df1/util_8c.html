◆ util_strsplit()

char ** util_strsplit ( const char *  string,
const char *  delimiter,
int  max_tokens 
)

Split string into substrings.

Split string into substrings on delimiter, returning at most max_tokens substrings, plus the remainder as a final substring if there is anything left in string. The delimiter is not among the substrings, except in the remainder if there is a remainder.

If string is NULL or "" NULL is returned. If \a delimiter isNULL` or "" an array of a single element is returned, with the single element being a copy of the full string. If max_tokens is reached the remaining characters in string are returned as the final array element. Consecutive instances of delimiter are treated as a single delimiter, there are no empty tokens returned.

Parameters
[in]stringstring to split on delimiter
[in]delimiterstring delimiter
[in]max_tokensmaximum number of substrings to generate
Returns
array of substrings, NULL-terminated
Note
free both the array and its elements with lib_free() after use
