Create a resource-bound spin button with custom range and formatting.

Create spin button that displays values between display_min and display_max but internally uses resource_min and resource_max for the resource. The display_format string is used to format the displayed value, and while the resource is an int the displayed value is a double, so the format string should reflect this. When displaying and setting the resource, scaling is applied to translate between the resource range and value and its displayed value and range.

Parameters
[in]resource_nameresource name
[in]resource_minresource minimum value
[in]resource_maxresource maximum value
[in]display_mindisplayed minimum value
[in]display_maxdisplayed maximum value
[in]display_stepdisplayed value stepping
[in]display_formatformat string for displayed value
Returns
