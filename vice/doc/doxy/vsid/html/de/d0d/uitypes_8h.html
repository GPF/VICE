Detailed Description

Gtk3 UI menu item types.

These types are shared between uimenu.{c,h} and uimachinemenu.{c,h}.

Due to someone splitting uimenu.c into uimenu.c and uimachinemenu.c we now have to deal with a lot of cross-dependencies between those files. So fixing that is definitely a TODO.

Menu item types.
