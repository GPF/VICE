GtkWidget * drive_dos_widget_create_combo ( int  unit)

Create combo box to select drive DOS expansion.

Create combo box that allows selecting a DOS expansion that is valid for the current unit's drive type.

Contains the following GObject keys:

  • UnitNumber: Drive unit number (8-11)
  • ModelDriveType: Drive type used for (re)construction of the model of the combo box, to avoid rebuilding the model on every sync() call
  • ChangedHandlerID: ID of the 'changed' event handler, used to block the changed event when calling sync()
Parameters
[in]unitdrive unit number (8-11)
