73/* From SCSI-1 standard:
74
75==============================================================================
76 Signal
77-----------
78MSG C/D I/O Phase Name Direction Of Transfer Comment
79------------------------------------------------------------------------------
80 0 0 0 DATA OUT Initiator to target \ Data
81 0 0 1 DATA IN Initiator from target / Phase
82 0 1 0 COMMAND Initiator to target
83 0 1 1 STATUS Initiator from target
84 1 0 0 *
85 1 0 1 *
86 1 1 0 MESSAGE OUT Initiator to target \ Message
87 1 1 1 MESSAGE IN Initiator from target / Phase
88==============================================================================
89
90*/
