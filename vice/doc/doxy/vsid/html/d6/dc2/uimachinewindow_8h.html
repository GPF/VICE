Detailed Description

Native GTK3 main emulator window code.

Author
Marcus Sutton logge.nosp@m.doub.nosp@m.t@gma.nosp@m.il.c.nosp@m.om

A "machine window" is any window whose primary purpose is to display an emulated screen. It usually also includes a menu bar and a status bar.
