mediator_t * mediator_new ( GtkWidget *  widget,
const char *  name,
GType  type 
)

Create and attach resource widget mediator.

Create new resource mediator object for resource name and attach it to widget. Registers a 'destroy' signal handler on widget that frees the mediator and its members.

The resource name can be NULL so the mediator can be used to wrangle additional state objects (like in cartwidgets.c).

Parameters
[in]widgetresource widget
[in]nameresource name
[in]typeresource type (G_TYPE_BOOLEAN, G_TYPE_INT or G_TYPE_STRING)
Returns
new resource mediator
