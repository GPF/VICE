int  unit,

Fill in a menu with controls for fliplist control.

Fliplist controls are placed at the end of a menu, after a separator. Any previously-existing fliplist controls within the menu will be removed.

Parameters
menuThe menu to be edited.
unitThe drive unit (8-11) that this menu will control.
separator_countThe number of menu separators in the part of the menu that does not involve the fliplist.
