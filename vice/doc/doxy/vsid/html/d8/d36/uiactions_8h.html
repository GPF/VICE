◆ IS_ACTION_NAME_CHAR

#define IS_ACTION_NAME_CHAR (   ch)
Value:
(isalpha((unsigned char)(ch)) || isdigit((unsigned char)(ch)) || \
ch == '_' || ch == '-' || ch == ':')

Check for valid action name character.

Check if ch is a valid character in an action name.

Supported characters are:

  • a-z
  • A-Z
  • 0-9
  • '_', '-' and ':'
