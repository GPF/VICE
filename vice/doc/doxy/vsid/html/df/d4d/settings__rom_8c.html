Detailed Description

Settings dialog to manage ROMs.

Presents a GtkListBox with expandable rows for machine, drive and drive expansion ROMs, using a resource file chooser widget for each ROM. Drag-n-drop should also work, though I've only tested that on my Debian box with Gnome so far.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl
