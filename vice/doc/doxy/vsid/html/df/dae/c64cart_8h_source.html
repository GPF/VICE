41/*
42 inputs:
43
44 !RESET C 6502 RESET pin(active low) buff'ed ttl out/unbuff'ed in
45 !IRQ 4 Interrupt Request line to 6502 (active low)
46 !NMI D 6502 Non Maskable Interrupt (active low) buff'ed ttl out, unbuff'ed in
47 !GAME 8 active low ls ttl input
48 !EXROM 9 active low ls ttl input
49 !DMA 13 Direct memory access request line (active low input) ls ttl input
50
51 outputs:
52
53 DOT CLOCK 6 8.18 MHz video dot clock
54 phi2 E Phase 2 system clock
55 BA 12 Bus available signal from the VIC-II chip unbuffered 1 Is load max.
56 R/!W 5 Read/Write (write active low)
57 !I/O1 7 I/O block 1 @ $DE00-$DEFF (active low) unbuffered I/O
58 !I/O2 10 I/O block 2 @ $DF00-$DFFF (active low) buff'ed ls ttl output
59 !ROML 11 8K decoded RAM/ROM block @ $8000 (active low) buffered ls ttl output
60 !ROMH B 8K decoded RAM/ROM block @ $E000 buffered
61
62 bidirectional:
63
64 D7-D0 14-21 Data bus bit 7-0 - unbuffered, 1 ls ttl load max
65 A15-A0 F- Y Address bus bit 0-15 - unbuffered, 1 ls ttl load max
66*/
