Detailed Description

Resource file chooser widget.

Presents a GtkEntry with an "open document" icon that serves as the "browse" button to open a file chooser dialog. By default no file matching pattern is used and no overwrite confirmation dialog is shown in case of the GTK_FILE_CHOOSER_ACTION_SAVE action.
