◆ action

int ui_menu_item_s::action

UI action ID as defined in uiactions.h.

The action ID is used for the hotkeys to be able to set and alter the hotkey assigned to the menu item.

Note
Must be unique or 0.
Do NOT use the same action ID for similar actions in different emulators, thanks to the run-time checking of the machine this will lead to the hotkeys code updating the first matching action it finds and ignoring the other actions with the same name.

◆ activate

bool ui_menu_item_s::activate

Use the 'activate' signal instead of the 'toggled' signal.

If a menu item is radio button, use the 'activate' signal instead of the 'toggled' signal so radio buttons that are already selected (e.g "Custom FPS" still activate their dialog when selected again. Has no meaning for other types of menu items.
