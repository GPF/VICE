Detailed Description

Numeric string connected to a resource.

Used for resources that store their numeric value as a string.

Supports using suffixes 'K', 'M', and 'G' for KiB, MiB and GiB respectively. Digits are accepted as well a navigation keys (left, right, home, end, insert, delete) and Tab, Return and Enter to accept input.
