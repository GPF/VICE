Set memory hack.

Pauses emulation when switching to another memory expansion hack to avoid having a running CPU access invalid memory. Unpauses emulation if the emulation wasn't already paused before switching memory expansion.

Parameters
[in]valuememory hack ID
[in]paramextra data (unused)
Returns
0 on succes, -1 on failure
