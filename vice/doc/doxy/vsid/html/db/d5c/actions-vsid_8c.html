Detailed Description

UI action implementations for VSID.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl
Note
This file cannot be used from ui.c since that causes massive linker errors due to the way vsid is linked. Currently registering the actions happens in vsidui.c, which magically does work.
