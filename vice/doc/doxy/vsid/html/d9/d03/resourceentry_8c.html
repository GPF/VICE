GtkWidget * vice_gtk3_resource_entry_new ( const char *  resource)

Create resource entry box that only reacts to 'full' entries.

Creates a resource-connected entry box that only updates the resource when the either the widget looses focus (due to Tab or mouse click somewhere else in the UI) or when the user presses 'Enter'. This behaviour differs from the other resource entry which updates its resource on every key press.
