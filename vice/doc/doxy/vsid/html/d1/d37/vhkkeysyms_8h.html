Detailed Description

UI-agnostic key symbols and names.

UI-agnostic key symbols and names - header.

List of keysym identifiers used by the hotkeys in VICE. Each UI toolkit has its own key symbols and names, so we need a way to refer to keys that isn't toolkit-specific.

The keysyms provided here are taken from /usr/include/X11/keysymdef.h.
