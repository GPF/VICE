Create printer driver selection widget.

Creates a group of radio buttons to select the driver of printer # device.

Uses a custom property "Driver" for the radio buttons and stores the resource name for device as the "ResourceName" property.

The following device lists are obtained dynamically: Printer 4/5 : [ascii, mps 801/802/803, cbm 2022/4023/8023, nl10, raw] Printer 6 : [1520, raw] Userport (3) : [ascii, nl10, raw]

Parameters
[in]devicedevice number (4-6) or 3 for userport
[in]callbackfunction to call on radio button selection (optional)
