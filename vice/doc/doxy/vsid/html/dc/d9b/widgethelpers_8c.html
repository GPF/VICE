Detailed Description

Helpers for creating Gtk3 widgets.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl

This file is supposed to contain some helper functions for boiler plate code, such as creating layout widgets, creating lists of radio or check boxes, etc.
