◆ vdrive_bam_get_track_entry()

static uint8_t * vdrive_bam_get_track_entry ( vdrive_t *  vdrive,
unsigned int  track,
unsigned int  sector 
)
static

Get pointer to the BAM entry for track in vdrive's image.

A CBMDOS BAM entry consists of a single byte indication the number of free sectors (unreliable since there's a lot of disks out there which set this to 0 to get a "0 blocks free."), followed by a bitmap of free/used sectors. The size of the bitmap depends on the image type, 1541 and 1571 have three bytes (enough for 21 sectors), while 1581 has 5 bytes (40 sectors).

[in]vdrivevdrive object
[in]tracktrack number
[in]sectorsector number ( needed for D9090/60)
