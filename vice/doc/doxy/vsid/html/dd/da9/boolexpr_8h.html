void bexpr_reset ( void  )

Reset tokenizer and evaluator for new expression.

Reset internal data structures for new expression, without freeing and then allocating resources again.

If using this module multiple times during the lifetime of a program (which is likely), do not call bexpr_free() followed by bexpr_init() when having to handle another expression, but call this function instead.
