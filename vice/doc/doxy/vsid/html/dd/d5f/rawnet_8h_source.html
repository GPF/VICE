51/*
52
53 These functions let the UI enumerate the available interfaces.
54
55 First, rawnet_enumadapter_open() is used to start enumeration.
56
57 rawnet_enumadapter() is then used to gather information for each adapter present
58 on the system, where:
59
60 ppname points to a pointer which will hold the name of the interface
61 ppdescription points to a pointer which will hold the description of the interface
62
63 For each of these parameters, new memory is allocated, so it has to be
64 freed with lib_free().
65 Note: The description can be NULL, since pcap_if_t.desc can be NULL, so
66 check the description before calling lib_free() on it.
67
68 rawnet_enumadapter_close() must be used to stop processing.
69
70 Each function returns 1 on success, and 0 on failure.
71 rawnet_enumadapter() only fails if there is no more adpater; in this case,
72 *ppname and *ppdescription are not altered.
73*/
