◆ on_frame_clock_update()

static void on_frame_clock_update ( GdkFrameClock *  clock,
gpointer  user_data 
)
static

Called once per frame for frequent asynchronous UI updates.

For example, vsid.c provides an updated play time from the VICE thread. It's not safe for the VICE thread to call GTK code - so instead we just store the value and update the UI each frame in here.
