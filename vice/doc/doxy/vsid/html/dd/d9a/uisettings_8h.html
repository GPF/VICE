typedef struct ui_settings_tree_node_s ui_settings_tree_node_t

Settings tree node object.

Contains name to display in the settings tree widget, an ID to allow for xpath-like access to tree nodes, a callback to render a widget next to the tree, and optionally a list of child nodes.
