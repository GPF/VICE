◆ set_speed_resource()

static void set_speed_resource ( int  speed)
static

Set "Speed" resource and update UI elements.

Set new value for "Speed" resource: postive values are CPU speed in percentage points (100 = 100%), negative values are FPS values (50 = 50Hz, 60 = 60Hz), using 0 means 100% CPU and machine FPS (dictated by the hardware).

Parameters
[in]speednew value for the resource
