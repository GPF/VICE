◆ resource_widget_set_string()

void resource_widget_set_string ( GtkWidget *  widget,
const char *  key,
const char *  value 
)

Set property key of widget to string value.

Sets key to value using g_object_set_data() with lib_strdup(), so this function can be called with temporary data from the caller. This function can be used to update key with a new value, the old string is freed before setting a new one.

Parameters
[in,out]widgetwidget
[in]keyproperty name
[in]valueproperty value
Note
Call resource_widget_free_string(widget, key) to free memory used by value.
