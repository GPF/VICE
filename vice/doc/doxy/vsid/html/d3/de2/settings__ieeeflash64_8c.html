Handler for the "toggled" event of the 'enable' check button.

Toggles the 'enabled' state of the IEEE Flash! 64 adapter/cart, but only if an EEPROM image has been specified, otherwise when trying to set the check button to 'true', an error message is displayed and the check button is reverted to 'off'.
