Detailed Description

Sanitize a filename for writing to the host OS.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl

Replace all occurences in a filename that would result in illegal tokens in the host OS' file system with underscores.

OS support:

  • Linux
  • Windows
  • MacOS
  • BeOS/Haiku (untested)
