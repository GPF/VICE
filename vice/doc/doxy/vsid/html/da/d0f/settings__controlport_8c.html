Add widgets for the joystick adapter ports.

Adds count widgets to select the emulated device for the adapter ports.

Only the widgets for active adapter ports (according to current configuration) will be shown. The function update_adapter_ports_visibility() is used when the control port devices change to update the adapter port widgets' visibility due to a changed configuration.
