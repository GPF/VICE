Detailed Description

Small LED widgets for the status bar.

Author
Bas Wassink b.was.nosp@m.sink.nosp@m.@zigg.nosp@m.o.nl

A small widget with a label and a drawing area displaying on/off status.

Can be used to display statuses like 'Pause' and 'Warp', and perhaps also key states such as 'ShiftLock', '40/80 Column' etc.

The widget is structured as follows:

+-GtkEventBox----------------—+ | +-GtkGrid–+-------------—+ | | | GtkLabel | GtkDrawingArea | | | +-------—+-------------—+ | +----------------------------—+
